# @stitches/astro

**@stitches/astro** is an Astro implementation of [stitches](https://stitches.dev), a CSS-in-JS library with a best-in-class developer experience.

<p><br /></p>

```sh
# with npm
npm install @stitches/astro

# with yarn
yarn add @stitches/astro
```

<p><br /></p>

For full documentation, visit [stitches.dev](https://stitches.dev).
